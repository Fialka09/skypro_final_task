Git

  • git branch — отображает список локальных веток.
  • git branch -a — показывает все ветки, включая удалённые.
  • git branch [name] — создаёт новую ветку с указанным именем.
  • git checkout [branch] — переключает на указанную ветку.
  • git log — выводит список коммитов от новых к старым с хешем, автором, датой и сообщением.
  • git blame [file] — показывает, кто и когда вносил изменения в каждую строку файла.
  • git status — показывает текущее состояние репозитория
  • git add [file] — добавляет конкретный файл в индекс (область подготовки к коммиту).
  • git commit -m "сообщение" — создаёт коммит с описанием изменений. Сообщение должно кратко и ясно отражать суть внесённых правок.
  • git commit --amend — редактирует последний коммит (например, чтобы исправить сообщение или добавить в него файлы).
